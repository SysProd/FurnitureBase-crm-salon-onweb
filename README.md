FurnitureBase-crm-salon-onweb
==
Internal CRM of the factory salon for receiving orders.

INSTALLATION
------------

### Install

To install the project, use the following command:
~~~
git clone https://github.com/SysProd/FurnitureBase-crm-salon-onweb
~~~
